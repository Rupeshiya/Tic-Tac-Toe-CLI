# Tic-Tac-Toe-CLI

A simple tic-tac-toe game using node.js command line application.

### Getting Started

Install `tic-tac-toe-cli` globally:

```
  npm i tic-tac-toe-cli -g
```
### Usage

Just run the cli on terminal:

```
  tic_tac
```
